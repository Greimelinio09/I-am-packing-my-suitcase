# 🧳 I am packing my suitcase

A simple C program that implements the classic memory game  
"I am packing my suitcase".

## 📖 Description

This project is a small console-based implementation of the well-known game:

> "I am packing my suitcase and I take with me..."

Each player has to repeat all previous items in the correct order and add a new one.  
The game tests memory and attention.

This version is written in C and runs in the terminal.

## 🎮 Features

- Add items to the suitcase
- Repeat previously entered items
- Memory-based gameplay
- Simple command line interface
- Lightweight and easy to run

## 🛠️ Technologies

- Language: C
- Standard libraries only (no external dependencies)
